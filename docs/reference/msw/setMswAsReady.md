# setMswAsReady

Indicates to the [AppRouter](../routing/appRouter.md) that [Mock Service Worker](https://mswjs.io/) is ready and the application can safely be rendered.

## Reference

```ts
setMswAsReady()
```

### Parameters

None

### Returns

Nothing

## Usage

```ts
import { setMswAsReady } from "@squide/firefly";

setMswAsReady();
```
