# useRoutes

Retrieve the registered routes from the `FireflyRuntime` instance.

## Reference

```ts
const routes = useRoutes()
```

### Parameters

None

### Returns

An array of `Route`.

## Usage

```ts
import { useRoutes } from "@squide/firefly";

const routes = useRoutes();
```
